# C++ Pac-Man

I created a console-based Pac-Man game implemented in **C++** using **object-oriented programming**, **grid-based movement**, and real-time keyboard input.

## Features
- Grid-based game board rendered in the console
- Player-controlled Pac-Man movement (WASD)
- Collectible pellets with score tracking
- Simple ghost AI with movement logic
- Collision detection (walls, pellets, ghosts)
- Game-over and win conditions

## Technologies Used
- **Language:** C++
- **Paradigm:** Object-Oriented Programming (OOP)
- **Tools:** Visual Studio, Git, GitHub
- **Platform:** Windows Console
